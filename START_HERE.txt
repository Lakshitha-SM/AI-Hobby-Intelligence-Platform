🔧 TROUBLESHOOTING:

If you get import errors:
    pip install --upgrade -r requirements.txt

If ChromaDB fails:
    Delete the chroma_storage/ folder
    The platform will rebuild it automatically

If semantic search is slow on first run:
    This is normal - the embedding model is being downloaded (~120MB)
    Subsequent runs will be much faster
